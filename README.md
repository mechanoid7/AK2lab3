```c++
--help=<arg> | -h<arg>
--print=<str> | -p<str> 
--creator | -c
--lab | -l
--easter | -e
```

help и print имеют опциональные аргуенты.

При передаче ключу --help/-h любой из доступных аргументов(-h, -p, -c, -l, -e и их длинные аналоги), выводит информацию о них.

При передаче ключу --print/-p аргумент - печатает его.
